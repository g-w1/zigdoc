Maps the address to its IPv6 equivalent. In most cases, you would want to map the address to its IPv6 equivalent rather than directly re-interpreting the address. 
pub fn mapToIPv6(self: IPv4) IPv6 {
    var octets: [16]u8 = undefined;
    mem.copy(u8, octets[0..12], &IPv6.v4_mapped_prefix);
    mem.copy(u8, octets[12..], &self.octets);
    return IPv6{ .octets = octets, .scope_id = IPv6.no_scope_id };
}
Directly re-interprets the address to its IPv6 equivalent. In most cases, you would want to map the address to its IPv6 equivalent rather than directly re-interpreting the address. 
pub fn toIPv6(self: IPv4) IPv6 {

    mem.set(u8, octets[0..12], 0);
    mem.copy(u8, octets[12..], &self.octets);
    return IPv6{ .octets = octets, .scope_id = IPv6.no_scope_id };
}

Returns whether or not the two addresses are equal to, less than, or greater than each other. 
pub fn cmp(self: IPv6, other: IPv6) math.Order {
    return switch (mem.order(u8, self.octets, other.octets)) {
        .eq => math.order(self.scope_id, other.scope_id),
        else => |order| order,
    };
}
