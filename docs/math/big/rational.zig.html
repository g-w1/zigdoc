An arbitrary-precision rational number. Memory is allocated as needed for operations to ensure full precision is kept. The precision of a Rational is only bounded by memory. Rational's are always normalized. That is, for a Rational r = p/q where p and q are integers, gcd(p, q) = 1 always. pub fn init(a: *Allocator) !Rational {
    return Rational{
        .p = try Int.init(a),
        .q = try Int.initSet(a, 1),
    };
}

Efficiently swap a Rational with another. This swaps the limb pointers and a full copy is not performed. The address of the limbs field will not be the same after this function. src
pub fn swap(r: *Rational, other: *Rational) void {
    r.p.swap(&other.p);
    r.q.swap(&other.q);
}

pub fn mul(r: *Rational, a: Rational, b: Rational) !void {
    try r.p.mul(a.p.toConst(), b.p.toConst());
    try r.q.mul(a.q.toConst(), b.q.toConst());
    try r.reduce();
}
