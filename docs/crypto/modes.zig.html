Counter mode. This mode creates a key stream by encrypting an incrementing counter using a block cipher, and adding it to the source material. Important: the counter mode doesn't provide authenticated encryption: the ciphertext can be trivially modified without this being detected. As a result, applications should generally never use it directly, but only in a construction that includes a MAC.
