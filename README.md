# zigdoc
zig -> docs

```
zigdoc help:
Example: `./zig-cache/bin/docgen ~/dev/zig/lib/std/ -url https://github.com/ziglang/zig/blob/master/lib/std`
Outputs do `docs` folder by default.
docgen FOLDER_LOCATION [ -json -o [output_folder] -url [source url] -h ]
-json for json output
-o for output folder
-url for source url
-h for this help menu
```
